Pairs Colorsheme, a color scheme with a unique touch.

==============================================================================
Color scheme options					Pairs-options*

						*g:Pairs_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:Pairs_transp_bg = 0
<
